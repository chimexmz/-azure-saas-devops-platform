# OJ Base-Farm Limited DevOps Platform

## 📌 Project Overview
OJ Base-Farm Limited is a cloud-native agriculture e-commerce platform designed to connect farmers and buyers with essential agricultural products.

The platform enables users to browse and purchase products such as fertilizers, manure, crop seeds, animal feeds, and farm equipment.

This project is built using ASP.NET Core Web API and follows a DevOps approach, focusing on automation, scalability, and maintainability through CI/CD, containerization, and cloud deployment on Microsoft Azure.

---

## 🎯 Objectives
- Build a scalable agriculture marketplace backend using ASP.NET Core
- Implement multi-currency support (EUR, USD, GBP)
- Apply clean and maintainable architecture principles
- Containerize the application using Docker
- Implement CI/CD pipelines using GitHub Actions
- Deploy infrastructure using Infrastructure as Code (Bicep/Terraform)
- Host the platform on Azure services (App Service / Container Apps)
- Implement monitoring, logging, and security best practices
